# Project Documentation 
# to run the project you need to run the following command in the terminal 
``` py py main.py  ```

## we have created manager for contrling database migration process
* to init your database we use 
``` py py manager.py db init   ```
* to make the migrations on the database 
``` py py manager.py db migrate ```
* to apply the migrations on the database 
``` py py manager.py db upgrade ```
